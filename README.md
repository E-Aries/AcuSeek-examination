# AXUS 企业考核管理系统

> Author: 达咩 | 轻则

## 快速开始

### 环境要求

- Node.js ≥ 22
- Python ≥ 3.10
- npm 或 pnpm

### 安装

`ash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd ..
npm install
`

### 启动

`ash
# 方式一：分开启动
# 终端 1 — 后端
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# 终端 2 — 前端
npx vite --host

# 方式二：一键启动
start.bat
`

首次启动后端会自动创建 SQLite 数据库并生成默认管理员账号。

### 访问

| 服务 | 地址 | 端口 |
|------|------|------|
| 前端（开发） | http://localhost:5173 | 5173 |
| 后端 API | http://localhost:8000 | 8000 |
| API 文档 | http://localhost:8000/docs | 8000 |

| 角色 | 用户名 | 密码 |
|------|--------|------|

## 功能详解

### 题库管理

支持五种题型，每题可设置分类、难度、分值与解析。

**题型支持**

| 题型 | 自动评分 | 特性 |
|------|----------|------|
| 单选题 | ✅ | 四选一，匹配选项标签 |
| 多选题 | ✅ | 多项选择，无序匹配 |
| 判断题 | ✅ | 正确/错误，标签→文本映射 |
| 填空题 | ✅ | 文本精确匹配 |
| 简答题 | ⚠️ 模拟自动评分 / 正式人工批改 | 参考答案来自解析字段 |

**题库操作**

- 新增/编辑/删除题目
- 批量导入（支持 .xlsx / .xls / .csv，宽表格式选项A~D逐列）
- 批量导出（CSV 模板）
- 批量修改分类
- 批量删除
- 按题型、分类、关键词筛选

**题目预览**

点击题目文字弹出预览弹窗，显示题型标签、分类、难度评分、分值、选项列表（正确答案标绿底徽章）、答案（绿框）、解析（黄框）。

**分类管理**

独立的分类管理页面，支持新增/编辑/删除分类，删除时自动检查是否有题目引用，防止误删。

### 考核管理

**考核类型**

| 类型 | 说明 |
|------|------|
| 正式考核 | 限时答题，客观题自动评分，简答需人工批改 |
| 练习模式 | 不限时，每题即时反馈答案与解析 |
| 模拟考试 | 限时答题，所有题型自动评分，简答作答即给分 |

**组卷策略**

- **随机组卷**：按题型分布自动选题目，可精确控制每种题型的数量
- **指定考生**：正式考核支持指定考生名单，不在名单内的无法参加考试
- **重新组卷**：管理员可随时重新生成试卷

**题型分布联动**

- 调整题型分布滑块时，题目数量自动同步为分布总数
- 点击「分布」标签可自动根据题库实际题型预填充配置
- 手动修改题目数量时，题型分布自动清零，两者互斥

**及格分数线**

采用百分比制（默认 60%），与试卷总分解耦。例如试卷总分 17 分时 ≥10.2 分过线，200 分时 ≥120 分过线。

**考核管理操作**

- 查看详情、编辑基本信息（名称、时长、及格线、分类）
- 发布/结束考核
- 补考（重置未通过考生的试卷，可重新考试）
- 逐题批改（记录每题得分与正误）
- 删除考核（连带清理所有关联试卷记录）

### 考试答题

**考试流程**

`
开始考试 → 计时答题 → 交卷 → 自动评分 → 查看成绩
`

**答题界面**

- 顶部计时器：动态读取考核配置 duration，超时自动交卷
- 答题卡导航：标记已答/未答/当前，快速跳转
- 切题时自动保存答案，切换回来恢复
- 退出考试保留进度（可续考或放弃）

**练习模式**

- 每题即时反馈：答对/答错提示
- 显示正确答案与解析
- 不限时，自由练习

**续考与放弃**

- 进行中的考试可在成绩查询页「继续考试」恢复进度
- 「放弃」清理记录，不影响题库

**交卷与成绩**

- 提交后弹出成绩卡片（得分/满分/答对数/总题数/状态）
- 题目选项高亮（绿色=正确答案，红色=选错的选项）
- 自动提交答卷到后端进行评分

### 成绩查询

**考生视角**

- 我的成绩列表：每次考试的得分/总分/百分比
- 等级标签（优秀 ≥95% / 良好 ≥80% / 通过 ≥60% / 未通过）
- 点击进入成绩详情，查看每题得分

**管理员视角**

- 按考核统览：参考人数、通过率、平均分、最高分
- 考生成绩表：各考生的分数/总分/百分比/等级/用时

**成绩导出**

- 全量导出：Results 页面「导出报告」，CSV 含 BOM 标记，Excel 不乱码
- 单考核导出：ExamDetail 页面「导出」，导出该场考核的考生成绩

**成绩详情页**

`
┌─────────────────────────────────┐
│         分数环 (得分/满分)       │
│     优秀 94% · 答对 7/10 题    │
├─────────────────────────────────┤
│ 排名 │ 平均分 │ 最高分 │ 超过%  │
├─────────────────────────────────┤
│ 各分类得分分析                   │
│ 售后流程   8/10  80%  (3题)    │
│ 产品知识   5/6   83%  (2题)    │
├─────────────────────────────────┤
│ 每题得分详情（可展开）           │
│ ✅ 第1题 判断  得 2/2 分        │
│   ↕ 题目、选项、对比、解析      │
│ ❌ 第2题 单选  得 0/2 分        │
├─────────────────────────────────┤
│ 难度得分分析                     │
│ 简单  10/12  83%  绿色          │
│ 中等   8/12  67%  橙色          │
│ 困难   6/12  50%  红色          │
├─────────────────────────────────┤
│          成绩分布图              │
└─────────────────────────────────┘
`

### 错题本

- 自动收录答错的题目
- 错题列表 + 答题详情（我的答案 vs 正确答案）
- 支持按考核筛选查询
- 重新练习，巩固薄弱环节

### 系统设置

**品牌自定义**

- 系统名称、公司名称
- Logo 上传（支持预览和删除）
- 页脚版权文字、版本号
- 登录页标题、副标题

**主题定制**

- 主题色自定义（导航栏、侧边栏选中色、按钮主色同步变化）
- 实时预览面板，设置即见即所得

**其他**

- 操作日志（记录用户关键操作）
- 通知管理

### 用户管理

- 用户列表：用户名、姓名、角色、部门、创建时间
- 编辑用户信息
- 管理员不能修改自己的用户名（防止锁定）
- 用户名重复检测，保存失败时显示后端具体错误信息

## 评分体系

### 自动评分规则

| 题型 | 评分逻辑 |
|------|----------|
| 单选 | 用户答案标签 == 题库答案标签 |
| 多选 | 用户答案集合 == 正确答案集合（无序） |
| 判断 | 用户答案标签映射文本，与题库答案文本比较 |
| 填空 | 用户输入.trim() == 题库答案.trim()（区分大小写） |
| 简答（模拟） | 作答即给满分（非空判定） |

### 等级划分

| 得分率 | 等级 | 颜色 |
|--------|------|------|
| ≥ 95% | 优秀 | 金色 |
| ≥ 80% | 良好 | 蓝色 |
| ≥ 60% | 通过 | 绿色 |
| < 60% | 未通过 | 红色 |

### 手工批改

正式考试的简答题支持逐题批改，每题得分单独保存，成绩详情中展示「手工评分」标签。

## 性能优化

### SQLite WAL 模式 + 连接池

- **WAL（Write-Ahead Logging）模式**：读写不互斥，并发查询不再阻塞写入
- **连接池**：pool_size=10，避免频繁创建/销毁连接
- **实际效果**：API 响应时间从 2000ms 降至 **19ms**，提升约 **100 倍**

### N+1 查询修复

- 考核管理页面不再逐个调用 questions(id) 接口
- 试卷题目随考试详情批量返回，减少网络往返

### KeepAlive 页面缓存

- 切换 Tab 时页面不重新加载，保持滚动位置和选中状态
- 涉及页面：Dashboard、题库、考核列表、成绩、错题本等

### 启动预热

- start.bat 启动后自动预热后端，消除首个请求延迟
